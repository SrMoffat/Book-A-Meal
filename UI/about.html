About Us

Book-A-Meal is an application that allows customers to make food orders and helps the food vendor know what the customers want to eat. The application users with admin credentials can manage meal options, setup menus for a specific day, see the orders made by other users, view the summary of daily earnings, and access order history.

A caterer can manage meal options (i.e: add, modify and delete), Setup menu for a specific day, See the orders made by the customers, See amount of money made by end of day, See order history, Add other caterers

A customer can create an account and log in, See the menu for a specific day, Select an option out of the menu. The customers can change their meal choice, see order history, and get notifications when the menu for the day has been set.
